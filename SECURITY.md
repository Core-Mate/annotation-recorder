# Security Policy

## Supported versions
This is currently a pre-1.0 project. Security fixes are provided on the latest `main` branch.

## Reporting a vulnerability
Please do not open public issues for sensitive security findings.

Report privately by email to: `security@example.com` (replace with your real contact).

Include:
- Affected commit/version
- Reproduction steps
- Impact assessment
- Any suggested remediation

We will acknowledge receipt within 7 days.
